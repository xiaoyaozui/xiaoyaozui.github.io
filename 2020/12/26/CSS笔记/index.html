属性选择器

选择含有指定属性的元素

[属性名]{}

选择含有指定属性和属性值的元素

[属性名=属性值]{}

选择含有指定属性和以指定值开头的属性值的元素

[属性名^=属性值]{}

选择含有指定属性和以指定值结尾的属性值的元素

[属性名%=属性值]{}

选择含有指定属性和含有指定值的属性值的元素

[属性名*=属性值]{}

伪类选择器

伪类描述一个元素的特殊状态

一般使用：开头

第一个子元素

:first-child{}

最后一个子元素

:last-child{}

第n个子元素

:nth-child(){}

特殊值：n 第n个；2n或even 偶数位；2n+1或odd 奇数位

child基于所有子元素中排序，of-type基于同类型子元素范围中排序

否定伪类

:not(){}

未访问过的链接

a:link{}

访问过的链接

a:visited{}

鼠标移入的状态

:hover{}

鼠标点击的状态

:active{}

::first-letter{}

::selection{}

继承的样式

长度单位

像素：px

百分比：相对于父元素的百分比

em：1em=1font-size

rem: 相对于根元素字体单位

颜色单位

RGB值

红绿蓝调配

RGB(数字,数字,数字) 范围0-255

RGBA值：A为透明度，范围0—1

十六进制的RGB值：#红绿蓝 范围：00-ff

HSL值

色相（0-360） 饱和度（0%-100%） 亮度（0%-100%）

文档流

网页的最底层

块元素

独占一行；默认宽度：父元素的全部；默认高度：内容撑开

行内元素

自左向右水平排列；默认宽度和高度均被内容撑开

盒模型

内容区；内边距；边框；外边距

内容区

大小由width,height属性设置

边框border

宽度(border-width)，默认三个像素，四个值（上右下左），三个值（上 左右 下），两个值（上下 左右），border-xxx-width(top right bottom left)；颜色(border-color)，默认值为color值（前景色）；样式(border-style)solid 实线 dotted 点状虚线 dashed 虚线 double 双线，默认值为null

简写：无顺序 空格隔开 border: border-top: …

内边距padding

一个盒子可见框大小包括内容区，边框，内边距

padding-xxx(top right bottom left)

外边距margin

影响盒子的位置

设置左上移动自己，设置右下移动别人

盒子的水平布局

auto ：width margin-left margin-ringht

width设置为auto，会调整为最大

盒子的垂直布局

overflow处理溢出（visible显示hidden裁去scroll两个滚动条auto需要的滚动条）

相邻垂直外边距发生折叠

兄弟元素：取两者之间的较大值；都为负值取绝对值最大；一正一负加和

父子元素：子会传递给父

行内元素的盒模型

不支持设置高度和宽度

padding,border,margin可设置，但不会影响垂直方向上的布局

超链接是行内元素

display

用来设置元素显示类型

inline 将元素设置为行元素

block 将元素设置为块元素

inline-block 将元素设置为行块元素，可设置宽高又不独占一行

table 将元素设置为一个表格

none 元素不在页面中显示，不占页面位置

visibility

visible 默认值，正常显示

hidden 隐藏，但占据页面位置
